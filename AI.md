# AI Basics

The training data is assumed to be constant while weights of a Network are said to be variable.

- Machine learning
- Supervised algorithms:
- Classification
- Regression
- Algorithms:
  - Linear Regression
  - Logistic Regression
  - SVM
  - Decision Tree
  - K-Nearest Neighbour
  - Random Forest
  - XG-Boost
 - UnSupervised algorithms:
   - CLustering

         - K-Means clustering
      - Density Based
   - PCA
   - Autoencoders
  - Evaluation metrics:
